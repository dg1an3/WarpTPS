# WarpTPS

> **This repository has been archived.** Development has moved to [dH](https://github.com/dg1an3/dH) where WarpTPS lives under the `WarpTps/` directory with its full commit history preserved.

---

**Interactive image warping and morphing using Thin Plate Spline transformations**

WarpTPS is a landmark-based image deformation tool that implements [Thin Plate Spline (TPS)](http://en.wikipedia.org/wiki/Thin_plate_spline) interpolation for image warping, morphing, and medical image registration visualization.
